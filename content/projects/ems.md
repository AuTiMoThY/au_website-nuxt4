<h3 class="font-h3">
專案概述
</h3>

使用 CodeIgniter 4 + Vue3 製作設備查詢介面，以多樣化圖表呈現數據，方便管理者更加快速的分析與判斷設備狀態

![每日用量熱圖](/images/projects/ems/heatmap.jpg)

<h3 class="font-h3">
開發過程
</h3>

- 該專案中擔任技術PM和前端開發
- 前期與客戶溝通、釐清需求，交付專案計畫書、時程表、驗收單等文件
- 撰寫API文件與後端工程師溝通資料傳遞格式
- API 串接使用 Axios，並對資料進行異常值處理、排除特定數值、數據重組
- 使用 ECharts.js 實現圖表繪製，如趨勢圖、熱力圖、折線圖、散佈圖
- 自定義可重用元件，如Vue composables(表單驗證、API請求、圖表繪製)、下拉選單(select2.js)、日期選擇(flatpickr.js)等
- 從前端開發、測試與除錯到上線共計六個月
