<h3 class="font-h3">
專案概述
</h3>

規劃與製作企業內部用的UI元件庫，將常用的元件或動態收集成一個站，方便設計師或前端工程師預覽與沿用

![每日用量熱圖](/images/projects/ui-library/index.png)

<h3 class="font-h3">
開發過程
</h3>

- 使用 Nuxt3、PrismJS 實現可預覽的元件和效果，以及展示程式碼
- 自定義模組化元件，如圖片、按鈕、卡片、輪播圖等元件
- 使用 GSAP 製作卡片翻轉、物件漂浮、滑鼠追蹤等動態效果
